# Mini Capital Market System

```bash

Client App
    ↓
Broker OMS
    ↓
Broker RMS
    ↓
Exchange Gateway
    ↓
Exchange RMS
    ↓
Matching Engine
    ↓
Market Data Feed

```

## Passive Order
- Jo order already order book me pada hua hai aur wait kar raha hai
- Liquidity provide karta hai

## Aggressive Order
- Jo naya aata hai aur immediately kisi existing order ko hit karta hai
- Liquidity consume karta hai
